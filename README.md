# 🍕 MigaOne — Pizza Store Website

A responsive, multi-page pizza restaurant landing page built with pure **HTML**, **CSS**, and **JavaScript**. Features a clean modern design with smooth navigation, an interactive menu carousel, product cards, customer reviews, animated stats counters, and a contact form.

---

## 🌐 Live Preview

Open `index.html` directly in your browser — no server or build step required.

---

## 🚀 Tech Stack

| Technology | Purpose |
|---|---|
| HTML5 | Page structure & semantic markup |
| CSS3 | Custom styling & responsive layout |
| JavaScript (Vanilla) | Interactive behavior & counters |
| [Bootstrap 5.3](https://getbootstrap.com/) | Grid system, carousel & responsive components |
| [Boxicons](https://boxicons.com/) | Icon library |

---

---

## 🎨 Sections

- **Navbar** — Fixed responsive navigation with hamburger menu on mobile (CSS-only toggle)
- **Hero** — Full-screen background image with restaurant tagline
- **About** — Creativity & design section with feature icons
- **Menu Carousel** — Bootstrap carousel showcasing Margherita, Sea Food, and Mix Cheese pizzas
- **Products** — Product cards grid with "Order Now" buttons
- **Happy Customers** — Customer satisfaction section
- **Reviews** — Customer testimonial with quote styling
- **Stats Counter** — Animated numbers for Customers, Lines of Code, and Projects
- **Contact** — Contact form with name, phone, email, and message fields

## ⚙️ Getting Started

No installation or build tools needed. Just clone and open:

```bash
# 1. Clone the repository
git clone https://github.com/Amratef0/MigaOne.git
cd MigaOne

# 2. Open in browser
open index.html
```

Or simply double-click `index.html` to open it in your default browser.

---

## 📱 Responsive Design

The website is fully responsive:

- **Desktop** — Full horizontal navbar with social media icons
- **Mobile (< 977px)** — Hamburger menu with slide-in navigation panel (CSS-only, no JavaScript required for toggle)
